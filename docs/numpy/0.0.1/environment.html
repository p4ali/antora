venv

python3.7 -m venv numpy-env
pip install -U numpy scipy matplotlib pandas ipython
source numpy-env/bin/activate
...
deactivate
You may need change ~/.pip/pip.conf to set your proxy if necessary. 
[global]
index-url = https://repository.example.com/repository/pypi-proxy/simple/
trusted-host = repository.example.com

IDE setup

  • Project Structure…​ > Platform Settings > SDKs, add a new SDK pointing to your venv.

  • Project Structure…​ > Project Settings > Project > Project SDK, new or add a SDK.
